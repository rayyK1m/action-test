# GOORM SW CAMP

## 🖥 Environment Support

- `node` : >=16

## 📦 Quick Start

```shell
# git-flow 설치 (ubuntu)
$ apt-get install git-flow

# git-flow 설치 (mac)
$ brew install git-flow-avh

# 패키지 설치 및 실행
$ npm install
$ npm run dev
```

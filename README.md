# GOORM SW CAMP

## :desktop_computer: Environment Support

-   `node` : >=16

## :package: Quick Start

#### GDS 세팅

[gds docs](https://gds.goorm.io/?path=/story/readme--page#-install)

#### 세팅 및 실행

```shell
# git-flow 설치 (ubuntu)
$ apt-get install git-flow
# git-flow 설치 (mac)
$ brew install git-flow-avh
# 패키지 설치 및 실행
$ npm install
$ npm run dev
```
